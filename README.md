AIM

To write a program for Frequency Modulation and Demodulation using SCILAB, and to observe and measure the frequency deviation and modulation index of FM.

EQUIPMENT REQUIRED
Computer with i3 Processor or higher
SCILAB Software
THEORY

Frequency Modulation (FM) is a modulation technique in which the frequency of a high-frequency carrier signal is varied in accordance with the instantaneous amplitude of the modulating (message) signal, while the amplitude of the carrier remains constant.

Frequency Deviation (Δf):
Frequency deviation is defined as the maximum shift in the carrier frequency from its unmodulated value due to modulation.

Modulation Index (mf):
The modulation index in FM is given by:
                       mf = Δf / fm
	​


where:

Δf = Frequency deviation
fm = Frequency of the modulating signal
FM GENERATION

Frequency modulation can be generated by varying the frequency of a carrier signal in proportion to the amplitude of the modulating signal. Practically, FM generation is achieved using methods such as direct and indirect modulation techniques.

ALGORITHM
Define Parameters:
Sampling frequency (Fs)
Signal duration (T)
Carrier frequency (Fc)
Modulating signal frequency (Fm)
Modulation index (β)
Generate Signals:
Generate the modulating signal (sinusoidal waveform)
Generate the carrier signal
Compute the FM modulated signal
FM Modulation:
The modulated signal is obtained by varying the carrier frequency according to the modulating signal
FM Demodulation:
Differentiate the FM signal to obtain frequency variations
Apply envelope detection (absolute value)
Use a low-pass Butterworth filter to recover the original signal
Visualization:
Plot the following signals:
Modulating signal
Carrier signal
FM modulated signal
Demodulated signal
PROCEDURE
Refer to the algorithm and write the SCILAB program.
Open SCILAB software on the system.
Create a new script file in the editor.
Enter the program code.
Save the file with an appropriate name.
Execute the program.
If any errors occur, debug and re-execute the code.
Verify the generated waveforms using plots and compare with the model waveform.
PROGRAM:

```
Ac = 7.9;
Am = 15.8;
Fm = 306;
Fc = 3060;
Fs = 30600;
B  =  3.5; 
T = 0:1/Fs:2/Fm; 
 

// Message signal
em = Am * cos(2*%pi*Fm*T);
subplot(3,1,1);
plot(T, em);
xtitle("Message Signal");
xgrid();

// Carrier signal
ec = Ac * cos(2*%pi*Fc*T);
subplot(3,1,2);
plot(T, ec);
xtitle("Carrier Signal");
xgrid();
// FM signal
efm = Ac * cos( (2*%pi*Fc*T) + (B * sin(2*%pi*Fm*T)) );
subplot(3,1,3);
plot(T, efm);
xtitle("FM Signal");
xgrid();
```
